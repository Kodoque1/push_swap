# Description:

The goal of this project is to provide for a given list of integers an efficient list of operations to sort them, the operations being stack based: push, swap, rotate and reverse rotate. We are limited to 2 stacks.

# Instructions:

* Compile the program via `make`.
* Launch the program with the command `./push_swap <list of integer>`
